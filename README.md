# Pktool
Pktoolはポケモンのレート対戦の技選択時間１分の間になるべく様々な情報を得ることを目標としたツールです。
とりあえずオフラインで動作するので速いです。

1. ダメージ計算
2. ポケモンのステータス算出
3. パーティメモ（WIP）

## インストール

```
gem install pktool
```
